# Weather App 

A modern, responsive weather application built with **React** and **Vite** that allows users to search for current weather conditions in cities around the world.

## üåü Features

- **Real-time Weather Data**: Fetches current weather information (Temperature, Humidity, Wind Speed) using the OpenWeatherMap API.
- **Dynamic Icons**: Displays different weather icons based on current conditions (Sunny, Rainy, Cloudy, Snowy, etc.).
- **Search History**: 
  - Automatically saves the last 3 searched cities.
  - View search history by clicking the search bar.
  - Click on a history item to quickly re-search.
  - Delete individual history items or clear all history.
- **Responsive Design**: Clean and accessible UI that works across devices.
